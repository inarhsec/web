Skills

My skill set spans network security implementation, security assessment, and governance, risk, and compliance (GRC), supported by continuous learning and hands-on experience across regulated environments.

Network Security & Architecture​

  • Network security design and segmentation
  • Firewall configuration and policy management
  • Secure routing, NAT, and VPN (IPSec / SSL)
  • Perimeter and internal network hardening
  • Secure branch, DC, and HQ network architecture
  • Zero Trust security concepts (learning & implementation-focused)

Platforms & Technologies

  • Palo Alto Networks
  • Fortinet (FortiGate, FortiManager)
  • Cisco (Switching, Routing, Security)
  • FatPipe SD-WAN
  • VLANs, ACLs, routing protocols

Vulnerability Assessment & Penetration Testing (VAPT)​

  • Network and infrastructure vulnerability assessments
  • Web application security testing (OWASP Top 10)
  • Configuration reviews and misconfiguration detection
  • Red Team methodology (learning & applied labs)
  • Report writing with remediation guidance

Tools & Techniques

  • Nmap, Burp Suite, Metasploit
  • Manual testing and validation
  • Risk-based vulnerability analysis
  • Secure configuration benchmarking

Governance, Risk & Compliance (GRC)​

  • Information Security Governance support
  • Control mapping and gap analysis
  • Audit preparation and evidence validation
  • Risk assessment and treatment planning
  • Policy, SOP, and compliance documentation

Frameworks & Standards

  • ISO/IEC 27001:2022
  • PCI-DSS
  • RBI Cyber Security Framework
  • NPCI guidelines (UPI, IMPS, ATM)
  • CIS Controls
  • NIST CSF & SP 800-series

Security Operations & Monitoring​

  • Log monitoring and analysis (SIEM concepts)
  • Incident response workflows (learning & practice)
  • SOC operations understanding
  • Alert triage and escalation models
  • Threat awareness and attack surface visibility

Systems & Platforms​

  • Windows Server & Active Directory fundamentals
  • Linux system administration basics
  • Secure server hardening
  • Patch and vulnerability management concepts
  • Endpoint security awareness (EDR/XDR basics)

Documentation & Communication​

  • Technical documentation & diagrams (HLD / LLD)
  • Audit-ready evidence preparation
  • Security assessment reports
  • Client-facing explanations (technical → business)
  • Knowledge sharing via notes and blogs

Development & Automation (Learning Phase)​

  • Bash scripting fundamentals
  • Python basics for security automation
  • API understanding for security integrations
  • Docusaurus documentation sites
  • Git & GitHub workflows

Soft Skills​

  • Analytical and problem-solving mindset
  • Strong attention to security detail
  • Continuous learner mentality
  • Collaboration with developers, auditors, and operations teams
  • Calm response during incidents and audits

Learning Mindset​

Cybersecurity is not static.
My skills evolve continuously through:

  • Hands-on labs and simulations
  • Real-world projects and audits
  • Reading advisories, standards, and post-incident reports
  • Teaching and documentation

Security skills are built through curiosity, discipline, and practice.
